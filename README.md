# Criar Site Simples (Fundação Bradesco)

## Descrição
Projeto desenvolvido para prática de criação de páginas web utilizando HTML e CSS.

## Tecnologias utilizadas
- HTML
- CSS

## Funcionalidades
- Estruturação de páginas web
- Estilização com CSS
- Organização de layout simples

## Como executar
Basta abrir o arquivo `index.html` em um navegador.

## Aprendizados
- Estrutura básica de páginas HTML
- Aplicação de estilos com CSS
- Organização de elementos na tela
